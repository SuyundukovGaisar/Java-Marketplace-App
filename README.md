# Каталог товаров (Маркетплейс)

Это консольное приложение на Java, реализующее базовую систему управления каталогом товаров для маркетплейса.

## Основные возможности

*   Регистрация и авторизация пользователей.
*   Просмотр каталога товаров.
*   Добавление, изменение и удаление товаров.
*   Фильтрация товаров по категории и бренду.
*   Кеширование запросов для ускорения повторного поиска.
*   Аудит действий пользователя.

## Технологии

*   Java Core
*   Java Collections Framework
